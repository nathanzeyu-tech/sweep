About us

Who we are

Sweep is a cleanup website for AYLUS, a nonprofit organization run by high school students. Our team focuses on environmental cleanliness, mainly across Oviedo and Winter Park right now, while building momentum for a wider community impact.

We make it easier for students, families, and local residents to see where help is needed, sign up for existing cleanup events, and recommend new places that deserve attention.

Impact

Visible progress, one cleanup at a time

Our impact is practical and local: cleaner sidewalks, safer parks, fewer loose plastics near waterways, and more young people taking ownership of the places they use every week.

More accessible service Clear locations and sign-up links make it easier for new volunteers to participate.
Student-led action High school volunteers plan, organize, and show up for hands-on cleanup work.

How it works

From cleanup idea to community action

01

Request or host

A student, club, or resident submits a cleanup idea with a location and details.

02

Review and approve

The Sweep team checks the request before it appears publicly on the calendar.

03

Volunteer signup

People choose an approved cleanup, sign up, and receive confirmation by email.

04

Track participation

Participant counts update so organizers can see interest and plan the event.

Built by students

The technology behind Sweep

Frontend

HTML, CSS, and JavaScript

The public site is built as a fast static web app with interactive map, calendar, forms, and responsive layouts.

Deployment

GitHub and Vercel

Code changes are uploaded through GitHub and deployed through Vercel so the site can stay live and easy to update.

Backend

Google Apps Script and Sheets

Host requests, approvals, volunteer signups, participant counts, and email notifications are connected through a Google Sheets backend.
